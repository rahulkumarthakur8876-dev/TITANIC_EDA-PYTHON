# Titanic Dataset - Exploratory Data Analysis (EDA)

## Overview
This repository contains a Python-based Exploratory Data Analysis (EDA) of the classic Titanic dataset. The project explores passenger demographics and uncovers patterns in survival rates using data manipulation and visualization libraries. 

## Tech Stack
* **Language:** Python
* **Libraries:** Pandas, NumPy, Seaborn, Matplotlib

## Key Insights
* **Survival by Gender:** Female passengers had a significantly higher survival rate (~74%) compared to male passengers (~18%).
* **Survival by Passenger Class:** First-class passengers had the highest survival rate (~62%), while third-class passengers had the lowest (~24%).
* **Demographics:** The majority of passengers were in the third class, and the majority of the overall passengers were male. 

## Files
* `Titanic.ipynb`: The Jupyter Notebook containing the data loading, exploration, statistical summaries, and visualizations.
